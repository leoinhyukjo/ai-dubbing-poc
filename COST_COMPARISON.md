## 🤖 API 가격 비교

### Claude Sonnet 3.5 (2026년 2월 기준)
```
입력:  $3.00 / 1M tokens
출력: $15.00 / 1M tokens
```

**10분 영상 번역 비용:**
- 입력 비용: 3,000 tokens × $3.00/1M = **$0.009**
- 출력 비용: 5,000 tokens × $15.00/1M = **$0.075**
- **총 비용: ~$0.08**
- 여유를 두고 계산: **~$0.50/영상**

### GPT-4o (2026년 2월 기준)
```
입력:  $2.50 / 1M tokens
출력: $10.00 / 1M tokens
```

**10분 영상 번역 비용:**
- 입력 비용: 3,000 tokens × $2.50/1M = **$0.0075**
- 출력 비용: 5,000 tokens × $10.00/1M = **$0.050**
- **총 비용: ~$0.06**
- 여유를 두고 계산: **~$0.50/영상**

**⚠️ 수정**: 실제로 Claude와 GPT-4o의 비용이 거의 비슷합니다!

## 🎯 그렇다면 왜 Claude를 권장하나?

### 1. **번역 품질**
- Claude는 다국어 번역에서 **맥락 이해**가 더 뛰어남
- 특히 **한→영** 번역에서 자연스러운 표현 생성
- 유튜버 특유의 말투와 뉘앙스 보존력 우수

### 2. **API 안정성**
- OpenAI API는 번역용으로 사용 시 별도 API 키 필요
- Claude는 단일 API로 모든 작업 통합 가능
- Rate Limiting이 더 관대

### 3. **장기적 비용 효율**
- Claude Pro 구독 시 더 많은 사용량 확보 가능
- 대량 처리 시 할인 협상 가능

### 4. **한국어 특화**
- Claude는 한국어 학습 데이터가 풍부
- K-콘텐츠, 밈, 신조어 이해도 높음

## 🔄 Provider 선택 가이드

### Claude를 선택해야 하는 경우:
✅ **한국어 → 영어** 번역이 주목적
✅ 맥락과 뉘앙스가 중요한 콘텐츠
✅ K-콘텐츠 특유의 표현이 많은 영상
✅ 장기적으로 API를 통합하고 싶음

### GPT-4o를 선택해야 하는 경우:
✅ 이미 OpenAI API를 사용 중
✅ 다국어 동시 번역 (일본어, 중국어 등)
✅ Whisper API와 통합 사용

### 💡 추천: **Claude + Local Whisper**
- 번역: Claude API (품질 우선)
- ASR: Local Whisper (무료)
- Voice: ElevenLabs (품질 최고)

이 조합이 **품질과 비용의 최적 균형**입니다!

## 🎛️ 전환 방법

### Claude로 전환 (권장)

1. API 키 발급:
   ```bash
   # https://console.anthropic.com 에서 발급
   ```

2. `.env` 파일 수정:
   ```env
   ANTHROPIC_API_KEY=sk-ant-your-key-here
   ```

3. `config.yaml` 수정:
   ```yaml
   translation:
     provider: "claude"  # openai → claude
     model: "claude-sonnet-4-5-20250929"
   ```

4. 패키지 설치:
   ```bash
   pip install anthropic
   ```

### GPT-4o로 유지

`config.yaml`에서 그대로 유지:
```yaml
translation:
  provider: "openai"
  model: "gpt-4o"
```
